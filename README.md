# Projeto Y.help

Plataforma de fácil acesso disponibilizada para o uso da comunidade em geral, podendo receber e oferecer doações com uma grande diversidade de materiais.
<br>
<br>
Atividade prática desenvolvida visando o aprendizado e aperfeiçoamento com HTML, CSS e JS.

# Objetivo

Solucionar uma demanda real da sociedade com base nos ensinamentos aprendidos ao longo do semestre.
